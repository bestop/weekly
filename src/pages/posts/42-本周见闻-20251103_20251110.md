## 实用工具
**1、wubh2012/navsite: 【水果导航】一个基于飞书多维表格数据的导航网站**

将飞书多维表格转成一个导航网站。



**2、dataease/SQLBot: 🔥 基于大模型和 RAG 的智能问数系统，对话式数据分析神器。Text-to-SQL Generation via LLMs using RAG.**

一个开源 Web 服务，只要配置大模型和数据源，就可以用自然语言查询数据源，实现 text2sql。



**3、MuseBot/README_ZH.md at main · yincongcyincong/MuseBot**

一个接入聊天软件的智能机器人，实现了 AI 对话与智能回复，支持多种大模型，可以接入 Telegram、飞书、钉钉、微信等平台。



**4、suitenumerique/docs: A collaborative note taking, wiki and documentation platform that scales. Built with Django and React.**

一个开源的内容管理系统，支持多人实时协作，需要自搭建。



**5、biliqiqi/baklab-web: Web frontend for BakLab.**

一个开源的社区平台，将论坛发现功能和实时聊天互动结合在一起，标题链接是前端仓库
